# sass-boilerplate
The boilerplate includes gulp, sass, sourcemaps, autoprefixer, breakpoint and browsersync. And nothing else.

## Requirements

**[Node.js](http://nodejs.org) v4.x.x - v6.9.x .**
**https://phoenixnap.com/kb/install-node-js-npm-on-windows**


## Quickstart
```bash
$ cd boilerplate
$ npm install
$ gulp
```
